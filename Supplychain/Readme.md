This project basically helps to track items in a supplychain network.
It has two smartcontract files, one is to create an Item in the supplychain and the other one is named as ItemManager which helps to track items in the supplychain and trigger the next steps in the supplychain.
It uses the hash generated after everystep to track the item in the supplychain.
This project was deployed and tested on ganache blockchain using truffle suite and it also has a simple frontend to it
